# Go and Gin Beginner Toolkit

## Description
A beginner friendly REST API built with Go and Gin to demonstrate basic API routing and configuration.
This project was built as part of a Generative AI learning capstone to demonstrate prompt-assisted learning.

## Requirements
- Go 1.20+
- Terminal
- Git

## Setup
1. Clone or unzip the project
2. Run `go mod tidy`
3. Run `go run main.go`

## Test
- http://localhost:8080
- http://localhost:8080/status
